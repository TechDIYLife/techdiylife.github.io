OpenAI最新开源模型 GPT-OSS 本地部署实测

1 概要

视频介绍：https://www.bilibili.com/video/BV1oBYrzVE2c/
实战教程与代码下载：下载链接

OpenAI最近开源了两款LLM，分别是 GPT-OSS-20B 和 GPT-OSS-120B。

关于此模型的几个关键点：

  1. 模型支持的能力全面： 出色的指令遵循能力，工具使用能力，基于CoT的深度推理能力，结构化输出。

  2. 模型的能力无法达到高能力（请参考 Introduction 中的介绍）

  3. 模型参数量分布对比（MLP部分占90%的参数）

  4. 量化后训练减少显存占用（120B模型只需80G，20B模型只需16G显存）

  5. MoE专家数量：128（120B） 和 32 （20B）， select top-4 experts for each token

  6. Tokenizer（o200k_harmony）,词表大小：201K token

  7. 训练时间：
    120B： 2百10万H100小时
    20B： 20万H100小时 （32台H100机器，训练1个月）

  8. 关于模型安全，幻觉等评价相关内容请参考技术报告

技术报告：https://cdn.openai.com/pdf/419b6906-9da6-406c-a19d-1bb078ac7637/oai_gpt-oss_model_card.pdf

内容包括：

  • 环境安装
